## PulseED – Anonymous Real-Time Academic Feedback Dashboard

This is a modern, responsive React frontend for **PulseED**, a smart campus app focused on anonymous real-time academic feedback.

### Tech Stack

- **React + Vite**
- **Tailwind CSS** (glassmorphism, gradients, responsive layout)
- **React Router v6**
- **Recharts** for analytics visualizations
- **Lucide React** for icons

### Pages

- **Student Feedback** – micro feedback buttons, optional text, toast on submit.
- **Faculty Dashboard** – stats cards, sentiment trend chart, difficulty heatmap, recent feedback.
- **Admin Analytics** – institution-wide stats, bar chart, risk badges, sentiment table.
- **Alerts** – flagged courses with issue type and suggested actions.

All data is static/dummy for now; there is **no backend**.

### Getting Started

```bash
pnpm install   # or npm install / yarn
pnpm dev       # or npm run dev / yarn dev
```

Then open the printed `localhost` URL in your browser.
